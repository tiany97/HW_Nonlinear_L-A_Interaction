# HW_Nonlinear_L-A_Interaction
All the code is run using Python 3.6.8

For European regional analysis: 

Run "New_YT_nonlinear_try_jja_detrend_WEU_submit_hpc_ERA5_land_Tmax_GBR_1950_5fold_simply_ERA5" to produce SHAP value

Run "G8_YT_nonlinear_XGBoost_WEU_C_original -ERA5_land_500time_ERAland_GBR_ver3.2_quantile" to access the nonlinearity of three contributors

Run "G8_YT_nonlinear_XGBoost_WEU_C_original -ERA5_land_500time_ERAland_GBR_ver3.2_CMIP6_model_SM_changing_rate" to access the model bias source

For global scale analysis:

Run "YT_nonlinear_try_jja_detrend_WEU_allgrid_jja_only_copy0_3d_MAE_ERAland_GBR_5fold_Tmax" to produce SHAP value

Run "YT_nonlinear_try_jja_detrend_allgrid_jja_only_draw MAE oringinal_abs_T_ERAland_5fold_Tmax_GBR_CMIP6_ML_Mollweide_quantile_regress_50_oldSig-main" to access the nonlinearity of three contributors
